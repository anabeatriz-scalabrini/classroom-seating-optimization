# classroom-seating-optimization
Scientific Initiation research on spatial combinatorial optimization using Python, Gurobi Optimizer, and local search algorithms.

## Repository Contents

* **`Dados/`**: Contains the dataset and the 30 computational instances representing different classroom configurations and social distancing constraints. 
* **`Modelo Exato/Codigo/`**: Contains the exact mathematical formulations implemented in Python using the Gurobi Optimizer. These scripts generate the optimal student seating and row layouts across the 4 evaluated scenarios.
* **`Heuristicas/`** *(Coming soon)*: Will host the implementations of the Iterated Local Search (ILS) and Adaptive Iterated Local Search with Path Relinking (AILS-PR) algorithms to compare performance against the exact models.
